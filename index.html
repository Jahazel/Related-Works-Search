Authors: Aditya Bodi, Pooyan Fazli, Shasta Ihorn, Yue-Ting Siu, Andrew Scott, Lothar Narins, Yash Kant, Abhishek Das, and Ilmi Yoon

Summary: This paper explores an AI-driven system that generates video descriptions for blind and low-vision users.

Low-Vision Reading Aids: Reading as a Pleasurable Experience

Author: C.M. Harrison

Summary: Discusses myReader, a redesigned reading aid for low-vision users that improves upon traditional CCTVs by reducing physical strain.

Questions: Have studies examined younger users? Was system training considered?

OCR with Text-to-Speech for Visually Impaired Users

Authors: Vijayanarayanan A, Savithiri R, Lekha P, Abbirami R S

Summary: Proposes a system integrating OCR with TTS to assist visually impaired users in reading printed or handwritten text.

Questions: How can it be adapted for structured PDFs? What UI strategies improve accessibility?

AI-Based Support System for Blind People

Authors: Rijwan Khan, Mohammad Husain, Jitendra, Mohammad Rashid Hussain, Mohammad Nadeem Ahmed, Arshad Ali

Summary: Describes a wearable device that helps blind users read documents and recognize objects through real-time audio feedback.

Portable Camera-Based Product Label Reading

Authors: Rajkumar N, Anand M.G, Barathiraja N

Summary: A camera-based text reading framework that isolates objects of interest and converts text to speech for blind users.

CueSee: Exploring Visual Cues for Low Vision Users

Authors: Yuhang Zhao, Sarit Szpiro, Jonathan Knighten, Shiri Azenkot

Summary: A headset-based system that highlights visual cues to assist low-vision users in tasks like locating items in a grocery store.

ImageAssist: Tools for Enhancing Touchscreen-Based Image Exploration

Authors: Vishnu Nair, Hanxiu ‘Hazel’ Zhu, Brian A. Smith

Summary: This research introduces a tool that allows BLV users to interact with images on a touchscreen, providing hints and descriptions as they explore.

State-of-the-Art

The latest AI-powered accessibility tools include wearable devices for real-time recognition, OCR-based systems for text extraction, and AI-driven solutions for digital document accessibility. While effective, many require specialized hardware or struggle with complex layouts. Our project aims to improve accessibility with a web-based solution that integrates both text and visual context without requiring additional devices.

Project Differentiation

Unlike existing technologies, our project focuses on creating a flexible web-based tool that processes various document types—including PDFs, handwritten text, and images—and converts them into speech. By avoiding specialized hardware, our approach ensures broader accessibility. Additionally, our system goes beyond text extraction by incorporating visual layout descriptions to enhance comprehension for low-vision users.
